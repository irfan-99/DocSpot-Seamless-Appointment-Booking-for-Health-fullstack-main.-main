# DocSpot-Seamless-Appointment-Booking-for-Health-fullstack-main.
DocSpot is a full-stack healthcare appointment booking web application developed using the MERN stack (MongoDB, Express.js, React.js, Node.js).

The platform allows patients to register, log in securely, book appointments with doctors, manage their schedules, and track appointment history. The system also provides administrative controls for managing users, doctors, and appointment records.
